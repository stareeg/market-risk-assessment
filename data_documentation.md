# Документация по данным (пункт 1 задания по рыночному риску)

Все данные собраны тетрадкой [`data_loader.ipynb`](data_loader.ipynb) напрямую из
первоисточников, **разрешённых в задании** (ЦБ РФ `cbr.ru` и Московская биржа
`moex.ru`), и сохранены в каталоге `data/` в формате **parquet**.

- **Период истории:** 2021-01-01 … 2026-01-01 (фактически торговые дни 2021-01-04 … 2025-12-30).
- **Воспроизводимость:** вся логика в `utils/`, seed зафиксирован (`utils/config.py: RANDOM_SEED=42`).
- **Запуск:** `jupyter nbconvert --to notebook --execute --inplace data_loader.ipynb`
  (или открыть тетрадку и выполнить все ячейки сверху вниз).

> Запас по количеству инструментов взят сознательно: где задание требует 5 облигаций —
> загружено **10**; где 10 акций — загружено **12**. Лишние инструменты можно отбросить
> на этапе формирования портфеля.

## 2. Происхождение данных (цепочка до первоисточника)

Задание требует уметь ответить «откуда данные?» и «а там они откуда?». Кратко:

- **MOEX ISS** (`iss.moex.com`) — это открытый API самой Московской биржи. Котировки
  (`history`) — это **результаты реальных биржевых торгов** (сделки участников торгов).
  Описания инструментов и расписания выплат — из **реестра инструментов биржи** (для ОФЗ
  первоисточник эмиссии — Минфин РФ, регистрирующий выпуск; биржа хранит параметры выпуска).
- **КБД ЦБ РФ** (`cbr.ru/hd_base/zcyc_params`) — ЦБ публикует параметры и значения
  бескупонной кривой, **построенной по методологии Московской биржи** (модель
  Нельсона—Сигеля—Свенссона) **по сделкам с ОФЗ на MOEX**. То есть первоисточник —
  снова биржевые сделки с ОФЗ, агрегированные в кривую.
- **Курсы валют ЦБ РФ** (`cbr.ru/scripts/XML_dynamic.asp`) — официальные курсы, которые
  ЦБ устанавливает ежедневно на основе данных валютного рынка (с июня 2024 г. — на основе
  внебиржевых данных, т.к. биржевые торги долларом/евро на MOEX прекращены).

## 3. Подробное описание файлов

### 3.1 `zcyc_cbr.parquet` — кривая бескупонной доходности (п. 1.a)

Безрисковые спот-доходности гособлигаций по фиксированным срокам.

| Колонка | Тип | Смысл |
|---|---|---|
| `DATE` | datetime | Торговый день |
| `0,25`, `0,5`, `0,75`, `1`, `2`, `3`, `5`, `7`, `10`, `15`, `20`, `30` | float | Доходность к погашению, **% годовых**, для срока в **годах** |

- **Что означает:** ставка непрерывного безрискового дисконтирования по сроку τ. Используется
  как риск-фактор для переоценки облигаций (п. 2, 4).
- **Источник:** `https://www.cbr.ru/hd_base/zcyc_params/` (параметр запроса — диапазон дат).
- **Важно:** ЦБ отдаёт уже готовые значения доходности по 12 узлам; промежуточные сроки
  при необходимости интерполируются. Кодировка страницы — UTF-8.

### 3.2 `bonds_descriptions.parquet` — карточки ОФЗ (п. 1.b)

10 выпусков **ОФЗ-ПД** (постоянный/фиксированный купон), подобранных под критерии задания.

| Колонка | Смысл |
|---|---|
| `NUMBER` | Короткий номер выпуска (26221 и т.п.) |
| `SECID` | Полный код инструмента на MOEX (SU26221RMFS0) |
| `ISIN`, `NAME` | ISIN и название |
| `MATDATE` | Дата погашения |
| `COUPONPERCENT` | Ставка купона, % годовых от номинала |
| `COUPONPERIOD` | Период купона, дней |
| `FACEVALUE` | Номинал (1000 руб.) |
| `TYPE` | Тип бумаги (`ofz_bond`) |
| `OFFERDATE` | Дата оферты — **пусто у всех** (бумаги без оферт) |
| `after_2026`, `no_offer` | Авто-проверка критериев задания (обе = True) |

**Отобранные выпуски** (покрывают кривую от короткого до длинного конца):

| № | SECID | Погашение | Купон, % |
|---|---|---|---:|
| 26219 | SU26219RMFS4 | 2026-09-16 | 7.75 |
| 26226 | SU26226RMFS9 | 2026-10-07 | 7.95 |
| 26207 | SU26207RMFS9 | 2027-02-03 | 8.15 |
| 26212 | SU26212RMFS9 | 2028-01-19 | 7.05 |
| 26224 | SU26224RMFS4 | 2029-05-23 | 6.90 |
| 26228 | SU26228RMFS5 | 2030-04-10 | 7.65 |
| 26218 | SU26218RMFS6 | 2031-09-17 | 8.50 |
| 26221 | SU26221RMFS0 | 2033-03-23 | 7.70 |
| 26225 | SU26225RMFS1 | 2034-05-10 | 7.25 |
| 26230 | SU26230RMFS1 | 2039-03-16 | 7.70 |

- **Почему именно ОФЗ-ПД:** только у них выплаты полностью известны заранее и не привязаны
  к показателям. Исключены: ОФЗ-ПК (плавающий купон ~ RUONIA), ОФЗ-ИН (индексируемый
  номинал ~ инфляция), ОФЗ-АД (амортизация). У ОФЗ-ПД нет оферт.
- **Источник:** `https://iss.moex.com/iss/securities/{SECID}.json` (блок `description`).

### 3.3 `bonds_coupons.parquet` — расписания выплат (п. 1.b)

Полное расписание купонов для каждого выпуска (история + будущие платежи).

| Колонка | Смысл |
|---|---|
| `NUMBER`, `secid`, `isin`, `name` | Идентификация выпуска |
| `coupondate` | Дата купонной выплаты |
| `value` | Размер купона в рублях на 1 облигацию |
| `valueprc` | Купон, % годовых |
| `facevalue`, `initialfacevalue` | Номинал на дату / первоначальный |
| `recorddate`, `startdate` | Дата фиксации / начала купонного периода |

- **Источник:** `https://iss.moex.com/iss/securities/{SECID}/bondization.json` (блок `coupons`).
  Амортизация (`amortizations`) и оферты (`offers`) проверены — у выбранных ОФЗ амортизации
  нет (единичное погашение номинала), оферт нет.

### 3.4 `bonds_history.parquet` — котировки ОФЗ (п. 1.c)

Дневная история торгов по основной доске ОФЗ **TQOB**.

| Колонка | Смысл |
|---|---|
| `TRADEDATE` | Торговый день |
| `SECID`, `NUMBER`, `SHORTNAME` | Идентификация бумаги |
| `CLOSE` | Цена закрытия, **% от номинала** (чистая цена) |
| `LEGALCLOSEPRICE` | Признанная цена закрытия, % от номинала |
| `ACCINT` | Накопленный купонный доход (НКД), руб. |
| `WAPRICE` | Средневзвешенная цена, % |
| `YIELDCLOSE` | Доходность к погашению на закрытии, % |
| `DURATION` | Дюрация, дней |
| `FACEVALUE`, `COUPONVALUE` | Номинал и текущий купон |
| `NUMTRADES`, `VOLUME`, `VALUE` | Число сделок, объём (шт.), оборот (руб.) |

- **Как считать «грязную» цену:** `(CLOSE/100)*FACEVALUE + ACCINT` — именно она нужна для
  рыночной оценки позиции в портфеле.
- **Источник:** `https://iss.moex.com/iss/history/engines/stock/markets/bonds/boards/TQOB/securities/{SECID}.json`.

### 3.5 `stocks_history.parquet` — котировки акций (п. 1.d)

12 «голубых фишек» с устойчивым тикером, доска **TQBR**, по 1271 торговому дню каждая.

Тикеры: `SBER, GAZP, LKOH, GMKN, ROSN, TATN, NVTK, MGNT, MTSS, PLZL, SNGS, CHMF`.

| Колонка | Смысл |
|---|---|
| `TRADEDATE`, `SECID` | День, тикер |
| `CLOSE` | Цена закрытия, руб. |
| `LEGALCLOSEPRICE` | Признанная цена закрытия, руб. |
| `WAPRICE` | Средневзвешенная цена, руб. |
| `OPEN`, `HIGH`, `LOW` | Цены открытия/макс/мин, руб. |
| `NUMTRADES`, `VOLUME`, `VALUE` | Число сделок, объём (шт.), оборот (руб.) |

- **Источник:** `https://iss.moex.com/iss/history/engines/stock/markets/shares/boards/TQBR/securities/{TICKER}.json`.
- **Замечание:** цены не скорректированы на дивиденды (это «грязные» рыночные цены). Для
  риск-факторов обычно используют логарифмические доходности `CLOSE`.

### 3.6 `indices_history.parquet` — индексы (п. 1.e)

| Колонка | Смысл |
|---|---|
| `TRADEDATE`, `SECID` | День; `IMOEX` или `RTSI` |
| `CLOSE`, `OPEN`, `HIGH`, `LOW` | Значения индекса (пункты) |
| `VALUE` | Оборот по бумагам индекса, руб. |

- **IMOEX** — индекс МосБиржи (рублёвый), **RTSI** — индекс РТС (тот же набор акций, но в
  долларах, поэтому несёт валютную компоненту).
- **Источник:** `https://iss.moex.com/iss/history/engines/stock/markets/index/securities/{SECID}.json`.

### 3.7 `brent_history.parquet` — нефть Brent (п. 1.e)

Спот Brent в открытом доступе MOEX/ЦБ не публикуют, поэтому построен **непрерывный ряд
фронт-месяца** из фьючерсов `BR` на срочном рынке FORTS.

| Колонка | Смысл |
|---|---|
| `TRADEDATE` | Торговый день |
| `FRONT_CONTRACT` | Код фьючерса, выбранного как фронт в этот день (напр. `BRN5`) |
| `BRENT_USD` | Цена закрытия фронт-контракта, **долл. США за баррель** |
| `VOLUME` | Дневной объём фронт-контракта (контрактов) |
| `OPENPOSITION` | Открытый интерес |

- **Логика «фронт-месяца»:** на каждый день берётся контракт `BR` с **максимальным дневным
  объёмом торгов** — это всегда ближний ликвидный фьючерс. При прокрутке (роллировании)
  контракта возможен небольшой скачок цены — это нормально для непрерывных фьючерсных рядов;
  при использовании как риск-фактора лучше работать с доходностями.
- **Источник:** `https://iss.moex.com/iss/history/engines/futures/markets/forts/securities/{SECID}.json`.

### 3.8 `fx_cbr.parquet` — курсы валют (п. 1.e)

| Колонка | Смысл |
|---|---|
| `DATE` | Дата установления курса |
| `CCY` | Валюта: `USD` или `EUR` |
| `RATE` | Официальный курс ЦБ за **1 единицу** валюты, руб. (нормирован на номинал) |

- **Источник:** `https://www.cbr.ru/scripts/XML_dynamic.asp?date_req1=..&date_req2=..&VAL_NM_RQ={код}`
  (USD = `R01235`, EUR = `R01239`).
- **Важная оговорка:** это **официальный курс ЦБ**, а не биржевой. Биржевые торги USD/EUR на
  MOEX прекращены с июня 2024 г., поэтому ЦБ-курс — единственный непрерывный источник на всём
  интервале. Для переоценки валютной позиции портфеля это допустимая и общепринятая база.

### 3.9 `forts_futures_2025-10-15.parquet` — фьючерсы на выбранный день (п. 1.f)

Все фьючерсы на доллар (`Si`), торговавшиеся **15.10.2025**, плюс пометка выбранного контракта.

| Колонка | Смысл |
|---|---|
| `SECID` | Код контракта (`SiZ5`, `SiH6`, …) |
| `CLOSE`, `SETTLEPRICE` | Цена закрытия / расчётная цена (руб. за 1000 USD) |
| `OPEN/LOW/HIGH`, `WAPRICE` | Цены за день |
| `VOLUME`, `OPENPOSITION`, `NUMTRADES` | Объём, открытый интерес, число сделок |
| `LSTDELDATE` | Дата экспирации контракта |
| `IS_CHOSEN_FRONT` | True у выбранного контракта |

- **Выбранный контракт:** **`SiZ5`** (экспирация 2025-12-18), `CLOSE = 80209`, `SETTLE = 80176`
  (т.е. USD/RUB ≈ 80.2). Выбран как ближайший фьючерс, до экспирации которого **более 1 месяца**
  от 15.10.2025 (ближний `SiU6`/недельные отброшены как слишком близкие).
- **Источник:** `https://iss.moex.com/iss/history/engines/futures/markets/forts/securities.json?date=2025-10-15&assetcode=Si`.

### 3.10 `forts_options_chain_2025-10-15.parquet` — опционная цепочка (п. 1.f, бонус)

Опционы на фьючерс `SiZ5` с экспирацией 18.12.2025 на день 15.10.2025: **93 страйка**,
по Call и Put на каждый (186 строк), страйки 64 000 … 130 000.

| Колонка | Смысл |
|---|---|
| `SECID` | Код опциона |
| `STRIKE` | Цена исполнения (в пунктах фьючерса Si) |
| `OPTIONTYPE` | `C` (Call) или `P` (Put) |
| `UNDERLYINGASSET` | Базовый фьючерс (`SiZ5`) |
| `LSTDELDATE` | Экспирация опциона (2025-12-18) |
| `CLOSE`, `SETTLEPRICE`, `WAPRICE` | Цены опциона (руб.) |
| `OPEN/LOW/HIGH`, `CHANGE` | Цены/изменение за день |
| `VOLUME`, `OPENPOSITION`, `NUMTRADES` | Объём, открытый интерес, число сделок |

- **Назначение:** для бонусного задания (оценка опционов — модель Блэка на фьючерс, калибровка
  по рыночной улыбке волатильности). Для основного задания достаточно фьючерса.
- **Источник:** `https://iss.moex.com/iss/history/engines/futures/markets/options/securities.json?date=2025-10-15&assetcode=Si`
  + описания контрактов `/iss/securities/{SECID}.json` (страйк/тип/базовый актив).

## 4. Известные оговорки и ограничения

1. **Курсы валют — официальные (ЦБ), не биржевые.** Биржевой USD/EUR на MOEX недоступен после
   июня 2024 г.; ЦБ-курс непрерывен на всём интервале. Это надо учитывать при трактовке
   валютного риск-фактора.
2. **Brent — фронт-месяц фьючерса, не спот.** При роллировании возможны технические скачки
   уровня; для риск-факторов используйте доходности, а не уровни.
3. **Цены акций не скорректированы на дивиденды/сплиты** — это сырые рыночные цены закрытия.
4. **КБД ЦБ** — модельная кривая (NSS) по сделкам с ОФЗ, а не «сырые» доходности отдельных бумаг.
5. **Календарь дат** у разных рядов слегка различается (праздники, разные рынки): акции — 1271
   день, КБД — 1256, индексы/облигации — свои. На этапе п. 2 ряды нужно выровнять по общим
   торговым датам.
6. **Запас инструментов:** загружены 10 ОФЗ и 12 акций; для основного портфеля задание требует
   5 и 10 соответственно — отберите нужные из загруженных.

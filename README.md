# Проектная работа №2. Оценка рыночного риска

Считаем рыночный риск (VaR и ES) портфеля из ОФЗ, акций и валюты. Проект это
сквозной пайплайн: сбор данных с российских площадок, выделение риск-факторов
через PCA, стохастические модели динамики факторов, оценка справедливой цены
инструментов, Monte Carlo для VaR и ES, бэктестинг и статистические тесты.
Учебный курс НИУ ВШЭ. Полное условие лежит в `project_tasks/task_description.md`.

## Портфель и дата оценки

- 5 ОФЗ (26219, 26207, 26224, 26221, 26230), по 10 млн руб в каждую.
- 10 акций (SBER, GAZP, LKOH, ROSN, TATN, NVTK, MGNT, MTSS, PLZL, CHMF), по 1 млн руб.
- Валюта: 100 млн руб в долларе и 100 млн руб в евро.
- Итого 260 млн руб. Риск считаем на 2 декабря 2025 г.

Состав задаётся в одном месте, в `config.py`. Период истории данных с 1 января
2021 г. по 1 января 2026 г.

## Как запустить

Поставить зависимости (лучше в отдельном окружении):

```bash
python -m venv .venv
python -m pip install -r requirements.txt
```

Запуск без аргументов прогоняет весь пайплайн по порядку:

```bash
python main.py                   # весь пайплайн
python main.py --stage data      # только сбор данных
python main.py --stage factors   # только риск-факторы
python main.py --list            # список этапов и их статус
```

Этапы идут строго по порядку: каждый читает parquet, сохранённый предыдущим.
Результаты пишутся в `data/`. Seed зафиксирован (`RANDOM_SEED` в `config.py`),
поэтому при повторном запуске числа совпадают.

## Как это работает по этапам

Этап 1, данные. Через открытые HTTP-запросы тянем с MOEX ISS котировки 10 ОФЗ,
12 акций, индексы IMOEX и RTSI, фьючерс и цепочку опционов на доллар, с ЦБ РФ
кривую бескупонной доходности и официальные курсы USD и EUR. Brent склеиваем в
непрерывный ряд из фьючерсов BR (на каждый день берём контракт с наибольшим
объёмом). Виден разрыв на остановке торгов MOEX в феврале-марте 2022, это
нормально.

Этап 2, риск-факторы. Чистим цены от сплитов, помечаем длинные календарные
разрывы, переходим к лог-доходностям и изменениям ставок в б.п. Сжимаем кривую и
акции через PCA, получаем 8 риск-факторов: RATE_PC1, RATE_PC2, RATE_PC3, EQ_PC1,
EQ_PC2, EQ_PC3, FX_USD, FX_EUR.

- Кривая: три компоненты дают 95.6% дисперсии (74.5 / 14.1 / 6.9), это уровень,
  наклон и кривизна.
- Акции: PC1 это рыночный фактор (54% дисперсии, corr с доходностью IMOEX 0.82),
  три компоненты вместе 66.3%. Около трети дисперсии это идиосинкразия отдельных
  бумаг, её возвращаем собственным шоком на этапе 5.
- Хвосты у всех факторов тяжёлые, уровни нестационарны, приращения стационарны.

Этап 3, модели. По каждому фактору подгоняем GARCH(1,1) с распределением
Стьюдента (тяжёлые хвосты), для совместной динамики корреляций ставим DCC. Всё
оцениваем методом максимального правдоподобия.

- Распределение Стьюдента побеждает нормальное у всех 8 факторов и по AIC, и по
  BIC. Среднее число степеней свободы 4.4, минимум 2.37 у евро (скачки рубля).
- Пять факторов вышли на a+b=1, это интегрированный GARCH (память волатильности
  как у RiskMetrics). Взрывной волатильности нет нигде.
- DCC: theta1=0.14, theta2=0.86, сумма 0.9965 меньше 1, корреляции устойчивы.

Этап 4, оценка. Облигации считаем дисконтированием будущих потоков по кривой ЦБ
(кубический сплайн по 12 узлам, эффективная годовая ставка). Акции через
факторную модель на трёх EQ-факторах плюс собственный остаток. Валюта это сам
курс.

- Облигации: ошибка модельной цены против рыночной RMSE 0.76%, MAPE 0.59%,
  максимум 1.27% на длинной 26230. НКД сходится с биржевым до копейки.
- Акции: R2 факторной модели от 0.55 до 0.98, в среднем 0.68. Оставшиеся около
  32% дисперсии это идиосинкразия, для неё считаем волатильность и хвост остатка.
- Валюта: переоценка точная по построению (на дату оценки USD 77.70, EUR 90.34).

Этап 5, VaR и ES. Методом Monte Carlo гоняем 50000 сценариев тех же 8 факторов
по модели GARCH-DCC-t на горизонты 1 и 10 дней. Начальное состояние берём строго
по истории до даты оценки. Каждый день обновляем условную дисперсию и корреляцию
из DCC, тянем коррелированные инновации Стьюдента с числом степеней свободы своим
у каждого фактора. По акциям добавляем независимый собственный шок, чтобы не
занижать их риск. Портфель ежедневно ребалансируем к целевым пропорциям. Считаем
VaR 99% и ES 97.5%.

- Весь портфель: горизонт 1 день VaR 99% 3.98 млн руб (1.53%), ES 97.5% 4.50 млн
  (1.73%); горизонт 10 дней VaR 99% 14.88 млн (5.72%), ES 97.5% 16.54 млн (6.36%).
- Основной риск в валюте (200 из 260 млн, очень тяжёлые хвосты). Облигации
  спокойные (1 день 0.49%), акции волатильнее (1 день 3.82%).
- Отношение VaR за 10 и за 1 день 3.74, чуть выше корня из 10, как и ожидали при
  персистентной волатильности.

Этап 6, бэктестинг. На каждый торговый день 2025 г. заново считаем VaR той же
моделью, что в этапе 5 (с датой прогноза на предыдущий день), и сравниваем с
фактической доходностью следующего дня. Пробой это день, когда убыток превысил
VaR. Параметры моделей оставляем оценёнными раз на всей истории, каждый день
заново гоним только фильтр волатильности до даты прогноза. Считаем по всему
портфелю и трём подпортфелям.

- Весь портфель: 5 пробоев из 250 (2.0% при ожидаемом 1%), модель чуть оптимистична
  на хвосте. Все 5 приходятся на дни, когда пробивает валюта.
- Облигации 0 пробоев (VaR консервативен), акции 2 (0.8%, норма), валюта 6 (2.4%).
- Пробои валюты это дни резкого укрепления рубля в 2025. Что риск портфеля
  определяет валюта, видно и тут: пробои портфеля совпадают с валютными.

Этап 7, тесты. По таблице пробоев из этапа 6 считаем три семейства тестов на
корректность VaR: Kupiec (доля пробоев), Christoffersen (независимость соседних
дней и условное покрытие) и Haas (по промежуткам между пробоями). Считаем по
всему портфелю и трём подпортфелям.

- Весь портфель проходит все тесты (Kupiec p=0.16, условное покрытие p=0.34),
  модель корректна.
- Облигации Kupiec отвергает (p=0.025), VaR слишком консервативен (ноль пробоев).
- Акции проходят покрытие (p=0.93). Haas отмечает, что два пробоя легли близко
  (промежуток 5 дней), но при двух пробоях это слабый сигнал.
- Валюта на грани по доле (p=0.059), но покрытие проходит. Это снова показывает,
  что риск портфеля определяет валюта.

Этап 8, бонус. Два дополнительных портфеля поверх основного.

Опционы на фьючерс Si оцениваем моделью Блэка-76 (опционы на фьючерс, а не на
спот). Опционы маржируемые, поэтому дисконтирования в формуле нет. Волатильность
калибруем по цепочке за 2025-10-15, на деньгах 20.1%, на крыльях видна улыбка.

- Колл и пут в деньгах: по своей волатильности страйка Блэк-76 точно повторяет
  биржевую цену (проверка реализации), по единой волатильности на деньгах
  расхождение -1.0% и -1.8% (вклад улыбки).

Облигации со встроенными опционами строим из ОФЗ 26230: вариант с офертой put и
отзывную call на 01.01.2026 по 100% номинала. Оцениваем коротконой моделью ставок
на биномиальной решётке BDT, калиброванной на кривую.

- Проверка: обычная облигация на решётке 63.4% совпадает с оценкой по кривой из
  этапа 4.
- Оферта put добавляет +36 п.п. (бумага глубоко ниже номинала, инвестор всегда
  предъявит к выкупу), отзыв call цену не меняет (вне денег).

Интеграция в основной Monte Carlo (этап встаёт в ту же схему): опционы через
фактор FX_USD и переоценку Блэком-76, облигации через PCA-сдвиги кривой и
пересборку решётки.
